Theory
Soit \[ E_{vide} \] l'énergie du vide de l'univers remplisse l'univers dans tous ses espaces à toutes les échelles.

Soit \[ i_{0}(t) \] le courant parcourant les électrodes du réacteur à plasma.

Soit \[ u_{0}(t) \] la tension entre les électrodes du réacteur à plasma.

Soit \[ \overrightarrow{B}_{0}(t) \] le champ magnétique émis par le courant \[ i_{0}(t) \].

Soit \[ \overrightarrow{E}_{0}(t) \] le champ électrique émis par la tension \[ u_{0}(t) \].

Il faut exprimer \[ \overrightarrow{B}_{0}(t) \] en fonction de \[ i_{0}(t) \] et \[ \overrightarrow{E}_{0}(t) \] en fonction de \[ u_{0}(t) \].

Soit \[ \overrightarrow{B}_{1}(t) \] le champ magnétique des aimants permanents centraux.

Soit \[ \overrightarrow{B}_{2}(t) \] le champ magnétique des aimants permanents externes.

Soit \[ \overrightarrow{B}_{01}(t) \] le champ magnétique résultant de l'intéraction entre le champ magnétique \[ \overrightarrow{B}_{0}(t) \] et le champ magnétique \[ \overrightarrow{B}_{1}(t) \].

Soit \[ \overrightarrow{B}_{012}(t) \] le champ magnétique résultant de l'intéraction entre le champ magnétique \[ \overrightarrow{B}_{01}(t) \] et le champ magnétique \[ \overrightarrow{B}_{2}(t) \].

Exprimer \[ \overrightarrow{B}_{01}(t) \] en fonction de \[ \overrightarrow{B}_{0}(t) \] et \[ \overrightarrow{B}_{1}(t) \].

Exprimer \[ \overrightarrow{B}_{012}(t) \] en fonction de \[ \overrightarrow{B}_{01}(t) \] et \[ \overrightarrow{B}_{2}(t) \] par susceptibilité magnétique et optique non linéaire.

Soit \[ I_{1}(t) \] le courant induit par le champ magnétique \[ \overrightarrow{B}_{012}(t) \] par induction électromagnétique.

Soit \[ L_{B} \] l'inductance de la bobine du générateur.

Soit \[ l_{B} \] la longueur de la bobine du générateur.

Soit \[ N_{B} \] le nombre de tours de fils de la bobine du générateur.

Soit \[ S_{B} \] la surface de la bobine du générateur.

Soit \[ \mu_{B} \] la susceptibilité magnetique du noyau de la bobine du générateur.

On a : \[ L_{B} = \frac{ \mu_{B} * N_{B}^{2} * S_{B} }{ l_{B} } \] la susceptibilité magnetique du noyau de la bobine du générateur.

Soit \[ E_{ma} \] l'énergie magnétique de la bobine du générateur.

Soit \[ P_{ma} \] la puissance instantanée de la bobine du générateur.

On a : \[ P_{ma} = \frac{d E_{ma}}{d t} \] la puissance instantanée de la bobine du générateur.

On a : \[ E_{ma} = \frac{ L_{B} * I_{1}(t)^{2} }{ 2 } \]

D'où, \[ P_{ma} = \frac{ L_{B} }{ 2 } * \frac{d I_{1}^{2}(t) }{d t } = L_{B} * I_{1}(t) * \frac{d I_{1}(t) }{d t } \]

Exprimer \[ I_{1}(t) \] en fonction de \[ \overrightarrow{B}_{012}(t) \]

Soit \[ \Phi_{B} \] le flux magnétique de la bobine du générateur.

On a : \[ \overrightarrow{B}_{012}(t) = \overrightarrow{B}_{0}(t) + \overrightarrow{B}_{1}(t) + \overrightarrow{B}_{2}(t) \]

On a : \[ \Phi_{B} = \iint \overrightarrow{B}_{012}(t) . \overrightarrow{dS}_{B}(t) = B_{012}(t) * S_{B} = L_{B} * I_{1}(t) \] \[ I_{1}(t) = \frac{ B_{012}(t) * S_{B} }{ L_{B} } \] \[ \frac{d I_{1}(t)}{d t} = \frac{ S_{B} }{ L_{B} } * \frac{d B_{012}(t) }{d t} \]

D'où, \[ P_{ma} = \frac{ L_{B} * B_{012}(t) * S_{B} * S_{B} }{ L_{B} * L_{B} } * \frac{d B_{012}(t) }{d t} = \frac{ B_{012}(t) * S_{B}^{2} }{ L_{B} } * \frac{d B_{012}(t) }{d t} = \frac{ l_{B} * S_{B} * B_{012}(t) }{ \mu_{B} * N_{B}^{2} } * \frac{d B_{012}(t) }{d t} \]

On a : \[ B_{0}(t) = B_{0} * \cos(\omega_{0} * t) \] \[ B_{01}(t) = B_{0} * \cos(\omega_{0} * t) + B_{1} \] \[ B_{012}(t) = B_{0} * \cos(\omega_{0} * t) + B_{1} + B_{2} \] \[ \frac{d B_{012}(t) }{d t} = - B_{0} * \omega_{0} * \sin(\omega_{0} * t) \]

D'où, \[ P_{ma} = \frac{ - l_{B} * S_{B} * ( B_{0} * \cos(\omega_{0} * t) + B_{1} + B_{2} ) * B_{0} * \omega_{0} * \sin(\omega_{0} * t) }{ \mu_{B} * N_{B}^{2} } \]

Soit \[ N_{r} \] le nombre de réacteur à plasma du générateur.

Soit \[ P_{ma7} \] la puissance du Pmeg-7.

D'où, \[ P_{ma7} = N_{r} * P_{ma} \] la puissance du Pmeg-7.

Soit \[ P_{0} = i_{0}(t) * u_{0}(t) \] la puissance d'entrée du réacteur à plasma.

Soit COP le coefficient de performance du générateur.

On a : \[ COP = \frac{ P_{ma} }{ P_{0} } \]

On a : \[ i_{0}(t) = I_{0} * \cos(\omega_{0} * t) \] \[ u_{0}(t) = U_{0} * \cos(\omega_{0} * t) \] \[ P_{0}(t) = I_{0} * U_{0} * \cos^{2}(\omega_{0} * t) \]

D'où, \[ COP = \frac{ - l_{B} * S_{B} * ( B_{0} * \cos(\omega_{0} * t) + B_{1} + B_{2} ) * B_{0} * \omega_{0} * \sin(\omega_{0} * t) }{ \mu_{B} * N_{B}^{2} * I_{0} * U_{0} * \cos^{2}(\omega_{0} * t) } \] \[ COP = \frac{ - l_{B} * S_{B} * ( B_{0} * \cos(\omega_{0} * t) + B_{1} + B_{2} ) * B_{0} * \omega_{0} * \tan(\omega_{0} * t) }{ \mu_{B} * N_{B}^{2} * I_{0} * U_{0} * \cos(\omega_{0} * t) } \]

Exprimer \[ B_{0} \] en fonction de \[ I_{0} \]

On a : \[ B_{0} = \frac{ I_{0} * \mu_{0} }{ \pi * d_{vis} } \] où \[ d_{vis} \] est le diamètre de la vis.

D'où, \[ COP_{r} = \frac{ P_{ma7} }{ P_{0} } = \frac{ N_{r} * P_{ma} }{ P_{0} } = N_{r} * COP \]

Soit \[ E_{0} \] l'énergie apportée par l'utilisateur au générateur.

Soit \[ E_{t} \] l'énergie totale du générateur.

Soit \[ \alpha \] le coefficient de capture du vide.

Soit \[ E_{cv} = \alpha * E_{0} \] l'énergie capturée du vide par le générateur.

On a : \[ E_{t} = E_{0} + E_{cv} = E_{0} + \alpha * E_{0} = E_{0} * ( \alpha + 1 ) \]

Soit \[ \eta \] le rendement de la machine.

On a : \[ \eta = \frac{ E_{0} }{ E_{t} } = \frac{ E_{0} }{ E_{0} * ( \alpha + 1 ) } = \frac{ 1 }{ ( \alpha + 1 ) } \]

On a : \[ COP = \frac{ - \mu_{0} * \omega_{0} * l_{B} * S_{B} * \tan(\omega_{0} * t) } { \pi * d_{vis} * \mu_{B} * N_{B}^{2} * U_{0} * \cos(\omega_{0} * t) } * ( \frac{ I_{0} * \mu_{0} * \cos(\omega_{0} * t) }{ \pi * d_{vis} } + B_{1} + B_{2} ) \]

On a : \[ P_{ma}(t) = \frac{ - I_{0} * \mu_{0} * \omega_{0} * l_{B} * S_{B} * \sin(\omega_{0} * t) } { \pi * d_{vis} * \mu_{B} * N_{B}^{2} } * ( \frac{ I_{0} * \mu_{0} * \cos(\omega_{0} * t) }{ \pi * d_{vis} } + B_{1} + B_{2} ) \]

On a : \[ I_{1}(t) = \frac{ S_{B} * ( B_{0} * \cos(\omega_{0} * t) + B_{1} + B_{2} ) }{ L_{B} } = \frac{ S_{B} * l_{B} * ( B_{0} * \cos(\omega_{0} * t) + B_{1} + B_{2} ) }{ \mu_{B} * N_{B}^{2} * S_{B} } = \frac{ l_{B} * ( B_{0} * \cos(\omega_{0} * t) + B_{1} + B_{2} ) }{ \mu_{B} * N_{B}^{2} } \]

On a : \[ U_{1}(t) = \frac{ P_{ma}(t) }{ I_{1}(t) } = \frac{ - I_{0} * \mu_{0} * \omega_{0} * l_{B} * S_{B} * \mu_{B} * N_{B}^{2} * \sin(\omega_{0} * t) * ( B_{0} * \cos(\omega_{0} * t) + B_{1} + B_{2} ) }{ \mu_{B} * N_{B}^{2} * \pi * d_{vis} * l_{B} * ( B_{0} * \cos(\omega_{0} * t) + B_{1} + B_{2} ) } \] \[ U_{1}(t) = \frac{ - I_{0} * \mu_{0} * \omega_{0} * \sin(\omega_{0} * t) }{ \pi * d_{vis} } \]
